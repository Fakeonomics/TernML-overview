Overview

TernML is a framework for building and training neural networks with discrete ternary weights {-1, 0, +1}. The quantization-aware training pipeline achieves a regularization-by-quantization effect — ternary accuracy exceeds the float baseline.

The framework supports both feedforward classification (TernMLayer) and graph-based reasoning (TernaT engine) with the same ternary weight technology.

Key Results

  • MNIST: 94.77% float → 96.15% ternary (TernMLayer 256→100→10)
  • Fashion-MNIST (CNN): 91.57% float → 92.02% ternary (conv32→64→FC128→10)
  • Fashion-MNIST (KAN): 83.03% float → 84.67% ternary (TernMLayer 256→100→10)
  • ELM mode: 99.3% of full backprop accuracy with zero hidden training
  • Graph-based reasoning: Scalable knowledge graph navigation with ternary controllers

Hardware Targets

  • Cortex-M0+ ($0.50 MCU) — 16 KB L1 — fits entirely
  • ESP32-S3 — 512 KB SRAM — with headroom
  • RISC-V GD32V — 32 KB — fits
  • Smartwatch DSP — 128 KB — with headroom

Technology

All operations are multiplexer-based: weight × activation = pass, negate, or zero. No DSP slices, no FPU required. Inference is pure add/shift — suitable for the cheapest microcontrollers.
